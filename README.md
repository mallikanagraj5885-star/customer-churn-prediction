#  Customer Churn Prediction

##  Overview
This project predicts customer churn using machine learning techniques. It helps telecom companies identify customers who are likely to leave.

---

##  Objective
To analyze customer data and build a predictive model to reduce customer churn.

---

##  Dataset
- Telco Customer Churn Dataset  
- 7043 customer records  
- 21 features including demographics, services, and billing  

---

##  Process
1. Data Cleaning & Preprocessing  
2. Handling missing values (TotalCharges)  
3. Encoding categorical variables  
4. Exploratory Data Analysis (EDA)  
5. Model Building:
   - Logistic Regression  
   - Random Forest  
6. Model Evaluation  

---

##  Results
- Logistic Regression Accuracy: **82%**  
- Random Forest ROC-AUC: **0.85**

---

##  Key Insights
- Customers with **low tenure** are more likely to churn  
- **Long-term contracts** reduce churn significantly  
- **Fiber optic users** show higher churn  
- **Electronic check payment** is linked to higher churn  
- Service-related features impact retention  

---

##  Business Recommendations
- Encourage customers to choose long-term contracts  
- Improve retention strategies for new customers  
- Enhance service quality for high-risk users  
- Promote automatic payment methods
